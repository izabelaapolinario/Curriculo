Jogo de Batalha Naval em C#

O jogo de Batalha Naval é uma versão eletrônica do clássico jogo de tabuleiro. Vamos descrever como funciona:

Configuração Inicial:

No início do jogo, os jogadores (computador e humano) selecionam as posições para suas embarcações. Cada embarcação ocupa um certo número de células no tabuleiro.
Os jogadores escolhem a linha e a coluna onde querem posicionar suas embarcações.

Tabuleiro:

O tabuleiro é uma matriz bidimensional (9x9).
As células são representadas por coordenadas (linha, coluna).
As embarcações são colocadas nas células do tabuleiro.

Rodadas:

Os jogadores alternam turnos para atacar.
O jogador humano escolhe uma célula para atacar (linha e coluna).
O computador também escolhe uma célula aleatória para atacar.

Resultado do Ataque:

Se a célula atacada contém uma parte de uma embarcação, é um acerto.
Se a célula está vazia, é um erro.
O jogo continua até que todas as partes de uma embarcação sejam atingidas (afundada) ou até que todos os ataques sejam feitos.

Fim do Jogo:

O jogo termina quando todas as embarcações de um dos jogadores forem afundadas.
O resultado é anunciado: vitória para o jogador que afundou todas as embarcações do oponente ou empate se ambos os jogadores ficarem sem embarcações.

Mensagem Final:

Ao final da rodada, o jogo exibe uma mensagem informando se foi um empate ou se houve um vencedor.
