Sistema com API de Filmes

Este projeto consiste em uma aplicação que utiliza uma API de filmes para exibir informações detalhadas sobre filmes. A interface permite que os usuários visualizem filmes em cards interativos com imagem, título e uma breve descrição. Ao clicar no botão "Abrir Filme", os usuários têm acesso a uma página com detalhes completos do filme, incluindo gênero, duração, elenco e muito mais. Todos os dados são consumidos a partir de uma API de filmes que fornece informações em formato JSON.
