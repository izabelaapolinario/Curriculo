Task Management Project

The main objective of the task management project is to help organizations and employees enhance their productivity. Through this application, it is possible to track deadlines, monitor task progress, and maintain high levels of motivation and focus. Additionally, it aims to reduce rework and prevent time waste, contributing to a more efficient organizational culture.

Specific Goals:


Employee Registration:
The platform allows companies to register their employees, creating a centralized and collaborative environment.
Task Registration:
The administrator can register tasks, defining deadlines and priorities.
Reward Catalog:
Employees can exchange virtual coins for rewards, encouraging task completion. The administrator sets the coin values and deadlines for each activity.
Transparency and Efficiency:
By integrating all these features, the application aims to improve task management, prevent financial losses, and promote a more effective and responsible organizational culture.
